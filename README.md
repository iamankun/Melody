# MELODY AI
Dự án dạy Melody AI cho tác vũ yêu thích cá nhân.
Để bắt đầu, hãy xem qua toàn bộ thư mục để xác định cấu trúc.

1. Cấu trúc dự án:
- Trang chính: Chứa các màn hình tạo thành trang chính
+ Trang thành phần bố cục: src/layout
+ Trang css: src/globals.css

- Thành phần ai: src/ai
+ Tạo ý tưởng, tạo ý tưởng âm nhạc, tạo code, tạo trình phát hoặc video,...
+ LLMs/ // Nơi để tôi thêm tài liệu học cho Melody AI
+ src/ai/record // Bản ghi tin nhắn, có cho phép Melody xem để cải thiện

- Thành phần app
- Thành phần component
- Thành phần contexts
- Thành phần hooks
- Thành phần thư viện
và một số phụ thuộc cơ bản khácc

2. Ngôn ngữ và ngôn ngữ
- Ngôn ngữ app: NodeJs, NextJs, TailwindCSS, TypeScript, React
- Ngôn ngữ thế giới: Vi-VN
